Installation

Up and running in seconds

Installer Script Recommended

Clone the repo once, then run the installer from any project. Picks up skills, prompts, and workflows.

# Clone once
git clone https://github.com/marcoacciarri/ai.git /tmp/ai

# From your project directory
/tmp/ai/install-agents

# Or selectively
/tmp/ai/install-agents skills prompts

Degit (no clone)

Pull files directly into your project's .github folder — no local clone needed.

# Everything
npx degit marcoacciarri/ai/.github .github

# Just skills
npx degit marcoacciarri/ai/.github/skills .github/skills
